# Image Placeholders

This directory contains placeholder images for the FeedMe application.

## Required Images

For the blog section to display properly, please add the following images:

1. `blog-hero-bg.jpg` - A large hero image for blog page banner (1920x600px recommended)
2. `meal-prep.jpg` - Image for the meal prep article (600x400px recommended)
3. `balanced-nutrition.jpg` - Image for the nutrition article (600x400px recommended)
4. `summer-ingredients.jpg` - Image for the seasonal ingredients article (600x400px recommended)
5. `japanese-cheesecake.jpg` - Image for the Japanese cheesecake recipe (600x400px recommended)
6. `apricot-chicken.jpg` - Image for the apricot chicken recipe (600x400px recommended)
7. `pepper-pasta.jpg` - Image for the pepper pasta recipe (600x400px recommended)

You can use stock photos or your own images. If you don't have images available, the application will fall back to a gradient display.

## Image Guidelines

- Use high-quality images in JPG or PNG format
- Keep file sizes reasonable (100KB-300KB per image is ideal)
- Use food photography with good lighting and composition
- For the hero image, use a darker or subtle image that won't interfere with the overlay text

- [Unsplash](https://unsplash.com/) - High-quality free photos
- [Pexels](https://www.pexels.com/) - Free stock photos and videos
